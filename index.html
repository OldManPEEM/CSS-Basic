หน่วย (unit)ใน CSS (Absolute) แบบตายตัว

px

Pixel (พิกเซล) เป็นหน่วยที่นิยมใช้มากที่สุด โดยที่ 1 px = 0.75pt สัมพันธ์กับรายละเอียดหน้าจอ

pt

Point โดยที่ 1pt = 1/72 inchs ใช้ในงานสิ่งพิมพ์

cm

เชนติเมตร

mm

มิลลิเมตร

in

inches(1in = 96px = 2.54cm)

pc

picas (1pc = 12pt)

หน่วย(Unit) ใน CSS (Relative) แบบอัตราส่วน

%

เป็นการกำหนดขนาดเป็นเปอร์เซ็นต์ มักใช้กับความกว้างหรือสูง

em
testem1

testem4

อ้างอิงขนาดกับ element parent ที่ใกล้ที่สุด ใช้ในการกำหนดจำนวนเท่าของ ขนาดปัจจุบัน เช่น หากขนาดที่ใช้ตอนนี้คือ 10 px

- ถ้ากำหนดขนาดเป็น 2em จะหมายถึง 2เท่าของขนาดปัจจุบัน คือ 20px

- ถ้ากำหนดขนาดเป็น 1.4em จะหมายถึง 1.4 เท่าของขนาดปัจจุบัน คือ14px

- ถ้ากำหนดเป็นขนาดเดิมก็กำหนดเป็น 1em = 10px

vw

1% หรือ 1/100 ของ viewport width

- width ของ Browser viewport เท่ากับ 750px ค่า 1vm = 7.5px

vh

1% หรือ 1/100 ของ viewport width

- height ของ Browser viewport เท่ากับ 900px ค่า 1 vh = 9px

vmin, vmax

กำหนดค่าต่ำสุด และค่าสูงสุดของ viewport
